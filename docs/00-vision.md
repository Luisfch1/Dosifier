# Visión

Concrete es una app para diseñar elementos de concreto reforzado partiendo de NSR-10 y usando ACI como respaldo
cuando existan vacíos/interpretaciones. El objetivo de V1 es **Vigas**, con experiencia interactiva:

- El usuario define geometría, apoyos y cargas.
- Se calculan V(x) y M(x) por combinación/envolvente.
- El usuario propone refuerzo y ve en vivo **Capacidad vs Demanda**.
- Se genera despiece y cuantificación (por grupos) con restricciones de longitud comercial de varillas.

Siguientes módulos: columnas, losas, cimentaciones, etc.
